# 🤖 Task Bot v2

Telegram-бот для управления задачами с дедлайнами, проектами, повторами и умными напоминаниями.

---

## 🚀 Запуск за 5 минут

### 1. Получи токен
1. Открой Telegram → **@BotFather** → `/newbot`
2. Скопируй токен вида `1234567890:AABBcc...`

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Запусти
```bash
BOT_TOKEN="твой_токен" python bot.py
```

---

## ☁️ Деплой на Railway (24/7)

1. Зарегистрируйся на [railway.app](https://railway.app)
2. **New Project → Deploy from GitHub** (загрузи файлы в репо сначала)
3. **Variables → Add** → `BOT_TOKEN` = твой токен
4. Railway найдёт `Procfile` и запустит бота автоматически

## 📋 Команды

### Добавить задачу
```
/add Название | Проект | Дедлайн | Приоритет | Повтор
```
Дедлайн: `20.03` · `20.03 18:00` · `+5д` · `завтра`
Приоритет: `high` · `normal` · `low`
Повтор: `daily 07:30` · `weekly 0 10:00` · `every 30 09:00`

Примеры:
```
/add Сдать отчёт | Работа | 20.03 18:00 | high
/add Зарядка | Здоровье | завтра 07:30 | normal | daily 07:30
/add Планёрка | Работа | 17.03 10:00 | high | weekly 0 10:00
/add Платёж ЖКХ | Финансы | +30д | normal | every 30 09:00
```
Для weekly: N = 0(Пн) 1(Вт) 2(Ср) 3(Чт) 4(Пт) 5(Сб) 6(Вс)

### Остальные команды
- `/list` — все задачи по проектам
- `/list Работа` — задачи проекта
- `/today` — дедлайн сегодня
- `/projects` — обзор проектов
- `/done 3` — выполнить (повтор создаёт следующую)
- `/del 3` — удалить
- `/remind 3 25.03 15:00` — своё напоминание
- `/settings` — настройки напоминаний
- `/settz Europe/Moscow` — часовой пояс
- `/setmorning 08:30` — время утреннего дайджеста

---

## ⏰ Напоминания

- 🌅 Утром в день дедлайна (настраиваемое время)
- 🔥 За 1 час до дедлайна
- 🔔 Своё время через /remind

Каждый тип можно вкл/выкл в /settings.

---

## 🔁 Повторяющиеся задачи

При /done автоматически создаётся следующая итерация:
- `daily HH:MM` — ежедневно
- `weekly N HH:MM` — еженедельно
- `every N HH:MM` — каждые N дней
